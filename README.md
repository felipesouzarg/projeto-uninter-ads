Atividade Extensionista 2: Tecnologia Aplicada à Inclusão Digital - Projeto
---
📋 Descrição do Projeto
Este projeto tem como objetivo capacitar mulheres empreendedoras em situação de vulnerabilidade, utilizando a linguagem Python como ferramenta de emancipação tecnológica. Através da automação de processos comerciais (como fluxo de caixa e registro de vendas), a iniciativa busca reduzir a exclusão digital e fortalecer a gestão de microempreendimentos locais.
🛠️ Tecnologias e Ferramentas
Linguagem: Python 3
Editor de Código: VS Code
Conceitos Aplicados: Lógica de programação, estruturas de repetição e automação de cálculos financeiros.
📂 Conteúdo do Repositório
Este repositório contém o script principal desenvolvido durante as oficinas, focado na praticidade e autonomia das participantes para a gestão de seus próprios negócios.
